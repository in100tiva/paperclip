<objective>
Implementar a fundação de propagação de locale do user → server context dentro do heartbeat. Este plano cria o helper `resolveRunOwnerLocale` que descobre qual idioma o operador humano que disparou o wake escolheu, e integra esse helper no heartbeat de modo que o resultado fique disponível em um campo `runtimeLocale` (string `"pt-BR" | "en-US"`) usado pelos Plans 11-02 (language directive) e 11-03 (skill variants + bundleKey extension).

**Purpose:** AGENT-SKILL-03 exige que a preferência de idioma do user (Better Auth `user.locale`, Fase 7) chegue ao spawn dos agentes sem acoplar a estado de cliente UI. Este plano resolve a peça server-side dessa propagação. Sem ele, Plans 11-02 e 11-03 não têm como saber qual locale aplicar.

**Output:**
- Helper puro testável `resolveRunOwnerLocale` em arquivo dedicado
- Suite unit GREEN cobrindo 4 cenários (sem wakeup, system actor, user pt-BR, user en-US)
- Heartbeat refatorado para chamar o helper antes de `adapter.execute` e expor `runtimeLocale` no contexto runtime que o adapter recebe
</objective>

    ```typescript
    import { eq } from "drizzle-orm";
    import { agentWakeupRequests } from "@paperclipai/db/schema/agent_wakeup_requests";
    import { authUsers } from "@paperclipai/db/schema/auth";
    import type { Database } from "@paperclipai/db"; // ou o type já usado em service layer — descobrir via grep antes

    export type RuntimeLocale = "pt-BR" | "en-US";

    function narrowLocale(value: unknown): RuntimeLocale {
      return value === "en-US" ? "en-US" : "pt-BR";
    }

    export async function resolveRunOwnerLocale(
      db: Database,
      wakeupRequestId: string | null
    ): Promise<RuntimeLocale> {
      if (!wakeupRequestId) return "pt-BR";
      const rows = await db
        .select({
          actorType: agentWakeupRequests.requestedByActorType,
          locale: authUsers.locale,
        })
        .from(agentWakeupRequests)
        .leftJoin(authUsers, eq(authUsers.id, agentWakeupRequests.requestedByActorId))
        .where(eq(agentWakeupRequests.id, wakeupRequestId))
        .limit(1);
      const row = rows[0];
      if (!row || row.actorType !== "user") return "pt-BR";
      return narrowLocale(row.locale);
    }
    ```
