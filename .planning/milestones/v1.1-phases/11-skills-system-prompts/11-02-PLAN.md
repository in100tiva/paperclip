<objective>
Implementar o conteúdo do **language directive** que é injetado nos system prompts dos agentes, e o ponto de injeção dentro de `agent-instructions.ts` + a cobertura de cache invalidation no `prompt-cache.ts` + o fallback para resume sessions claude-local.

**Purpose:** AGENT-SKILL-01 exige que o system prompt carregue uma instrução condicional de idioma. Sem ela, mesmo com locale propagado (Plan 11-01), o modelo continuará respondendo em inglês porque nada no prompt diz para falar pt-BR. Este plano resolve a peça "qual é o texto e onde ele entra".

**Output:**
- Função pura `buildLanguageDirectiveBlock` com texto exato do directive
- Hook em `agent-instructions.exportFiles` que anexa o bloco quando `locale === "pt-BR"`
- `bundleKey` inclui locale (cache invalidation correto — Pitfall 1)
- Fallback em `execute.ts` para resume sessions (Pitfall 2 — directive via prompt body)
- Suites unit + integration GREEN
</objective>
